cd 프로젝트 폴더 경로
프로젝트 폴더로 이동

git clone
.git 파일 만들기

-----------------------------------------

git config --global user.name "이름"
유저 이름 설정

git config --global user.email "이메일"
유저 이메일 설정. 깃허브 사용 시 입력한 이메일.

git config --list
정보 제대로 들어갔는지 확인

git init
초기화

add .
해당 폴더에 있는 파일을 모두 올리겠다.

add a.txt
a.txt 파일을 올리겠다

git status
상태 확인. 파일에 어떤 내용물이 들어있는지 알려주는 것인 듯.

git commit -m "히스토리명"
히스토리를 만듦

git remote add origin 깃 주소
레포지토리에 연결

git remote -v
잘 연결됐는지 확인

git branch
branch명 확인

git push origin 브랜치명
해당 브랜치에 파일 업로드

------------------------------------------

.git이 들어있는 프로젝트 폴더 이름과 깃허브의 레포지토리 이름이 같아야 하는가?
